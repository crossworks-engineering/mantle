# 0.23.1 — June 12, 2026

## Highlights

- **This changelog.** Mantle now has a dedicated **/changelog** page — every release worth announcing gets a readable entry like this one, with previous versions linked below. You're reading the first.
- **"What's new?" in the sidebar.** The version number now lives at the foot of the sidebar and links here. After an update, it carries a **What's new?** pill until you've had a look — then it quietly steps aside until the next release.

## Notes

- Read state is per browser (localStorage), so a fresh device shows the pill once until you visit the changelog there too.
