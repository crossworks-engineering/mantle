# 0.23.0 — June 12, 2026

## Highlights

- **Security hardening across the API Console and agent autonomy.** A full multi-agent security audit of the v0.21–0.22 tool-building work, with every finding fixed:
  - **Secrets stay out of the model.** Tool responses are scrubbed before the model ever sees them — a service that echoes your API key back no longer leaks it into the conversation (error messages and base64-encoded forms included).
  - **`web_fetch` can't reach your internal network.** Private, loopback, link-local, and cloud-metadata addresses are blocked, and the guard is re-checked on every redirect hop.
  - **Redirects drop credentials.** Cross-origin redirects no longer carry secret-bearing headers along for the ride.
  - **Agents can't promote themselves.** Tool-group grants refuse shell tools and self-grants, and an agent can't disarm a confirmation gate on an existing tool.
- **New setting: "Require my approval for agent-built tools"** (Settings → Tools). When on, any tool an agent authors starts confirmation-gated, and only you can clear the gate.
- **Prompt-injection containment.** Everything retrieved from your brain — facts, search hits, passages — is fenced as *data, never instructions*, on both chat turns and unattended heartbeat runs. And `email_send` now fails closed: with no contacts, agents can only email you.

## Notes

- 1,392 tests green after the pass; the audit and fixes are documented in the security docs.
