# 0.39.0 — 2026-06-23

Point Appsmith at a specific part of your app: click a region in the preview to focus the next edit there.

## Highlights

- **Select-to-focus in the app builder.** The Builder tab has a new **Select
  element** toggle. Turn it on, hover the live preview, and the area under your
  cursor outlines; click to lock it in (click the same one again to clear). The
  Assist panel then shows a *"Focusing …"* chip, and your next request tells
  Appsmith to change **only** that region and leave the rest of the app
  untouched. `Esc` exits select mode; the lock survives so you can type.

- **Works on every app, no rebuild.** The selection overlay is injected by the
  host into the preview sandbox — it isn't part of the app's own code — so it
  lights up existing apps immediately. It highlights the regions Appsmith marks
  while building (`data-app-region`); single-select by design (pick the outer
  region to cover more).

## Notes

- No schema migration in this release.
- The focus mechanism reuses the existing `ai-assist` `focusRegionIds` contract;
  this release adds the missing front-end (the in-preview inspector + the toggle).
