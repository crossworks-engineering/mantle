# 0.113.0 — 2026-07-03

**Name your brain in the header.** A new **Site name** field in
Settings → Profile replaces the top-left "mantle" wordmark with your own
label — e.g. "Natref" — so when you run several brains it's obvious at a
glance which one you're looking at. Cosmetic only: leave it blank to keep
the Mantle wordmark, and the version badge stays either way.

- Stored as a profile preference (`siteName`, 40-char cap, trimmed) with a
  shared read/write projection so it can't drift between save and load.
- Served to the app shell via `/api/shell`; the header updates immediately
  after saving, no reload needed.
