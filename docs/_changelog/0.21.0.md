# 0.21.0 — June 11, 2026

## Highlights

- **API Console — a built-in Postman for Mantle** (sidebar: System → **API Console**). One searchable screen that explores and runs all three of Mantle's call surfaces:
  - **Built-in API** — a 142-endpoint catalog of every REST route, grouped by resource, with query-param docs, body examples, and `{param}` path chips. Same-origin calls ride your session; secret-bearing calls go through a server proxy.
  - **Built-in MCP** — listed live from the real MCP server (zero catalog drift) and invokable in place.
  - **Agent tools** — every registered tool, run through the real dispatcher, so a console test exercises exactly what an agent call would.
- **"Save as agent tool"** turns any HTTP request into a reusable agent tool. Templates support `{param}` placeholders filled from model input and `{{secret:service/label}}` references that decrypt from the key vault at dispatch — plaintext never reaches the model or the browser. Granted via tool groups, so heartbeat routines pick new tools up automatically.

## Notes

- Search matches parameter names (`{id}`, `agentSlug`, …) across all groups.
- Saved requests and history persist locally with tokens blanked.
- Settings → Tools gained header/query/body template editors.
