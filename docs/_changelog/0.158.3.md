# v0.158.3 — app_db_seed, and a Stop button on a running turn

Two pieces of work that had been finished since 2026-07-20 and sitting
unmerged. Nothing new was written for this release; it is the merge.

## Added

- **`app_db_seed`** — bulk reference-data loading for the mini-app builder, so
  an app that needs a lookup table doesn't have to be seeded a row at a time
  through `app_db_schema_set` + hand-written inserts.
- **Stop button on a running turn's Journey page.** A turn you can watch is a
  turn you should be able to end; previously the only way out of a long or
  wedged turn was to leave it running.

## Fixed

- **`app_build` failures now fail the tool call.** A broken build reported
  success, so the model went on believing it had shipped a working app and the
  failure surfaced later, somewhere less obviously connected to the cause.
