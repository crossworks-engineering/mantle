# 0.29.0 — 2026-06-22

Smarter reminder delivery, and updates that seed themselves.

## Highlights

- **Reminders follow the channel you actually use.** Event reminders now go to
  wherever you last talked to Mantle — your Telegram bot DM or a push to the
  companion app — tracked automatically as you message, and overridable in
  **Settings → Profile**. A manual choice holds until your next message on the
  other channel. (See `docs/reminder-delivery-routing.md`.)

- **Push quiet hours removed.** The companion's old in-app quiet-window is gone —
  your phone's OS-level Do Not Disturb already covers night-time muting, so the
  send-worker no longer carries a duplicate setting. *Migration 0095 drops the
  now-unused `push_prefs` quiet-hours columns.*

- **Updates seed themselves.** Pulling a new image and bringing the stack up now
  reconciles your brain to the manifest automatically on web boot — new API
  tools, new skills, and tool-group membership land **without running any
  `seed:*` script**, including for self-hosters who just
  `docker compose pull && up -d`. Runs once per version, only adds (never removes
  a grant), and is opt-out via `MANTLE_DISABLE_BOOT_RECONCILE=1`. This is what
  makes 0.28.0's maps/directions + timezone tools reach an upgraded brain on their
  own.

## Notes

- First deploy carrying the boot reconcile: on upgrade, the web server syncs the
  manifest once and records the version, so subsequent restarts are no-ops.
