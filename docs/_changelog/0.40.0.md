# 0.40.0 — 2026-06-23

The app builder's Code tab is now editable — tweak the source yourself, format it, save, and build.

## Highlights

- **Edit app code directly.** The Code tab's viewer is now a lightweight editor:
  type into any file with live syntax highlighting (a transparent textarea over
  the highlighter — no heavy editor dependency), Tab to indent. A dirty dot marks
  unsaved changes; **Save** writes them to the draft, and **Build** saves first if
  needed, so what you compile is always what you see.

- **One-click formatting.** A **Format** button pretty-prints the current file
  with Prettier (TS/TSX, JS/JSX, CSS/SCSS/LESS, JSON, HTML, Markdown). Formatting
  runs server-side, so the parser plugins stay off the browser bundle.

## Notes

- Edits land in the **draft**, like Appsmith's — review the preview and Publish
  when ready; Discard reverts. Running Appsmith (Assist) or a build re-syncs the
  editor from the server and drops unsaved local edits, so Save before you hand
  off to the agent.
- No schema migration in this release.
