# 0.101.0 — 2026-06-30

**Thinking, now a per-user choice.** Whether the assistant reasons before it
answers is no longer a per-box environment switch — it lives in your own profile,
next to the live-thinking toggle, and applies wherever you talk to your brain.

## Highlights

- **Thinking budget in Settings → Profile.** Under **Live thinking & streaming**
  there's a new **Thinking budget** — Off / Low / Medium / High. Real model
  reasoning is requested only when the switch is on *and* a budget above Off is
  set, so nothing changes unless you opt in. It applies the same way across the
  web assistant, Telegram, and unattended heartbeats, and a specialist the
  assistant delegates to inherits the same setting.

- **Safe by construction.** The budget is automatically kept within the model's
  reply allowance, so a high setting can't crash a turn — large budgets are
  trimmed to leave room for the answer, and when there isn't room the reasoning
  is simply skipped rather than failing. This replaces the old
  `MANTLE_THINKING_BUDGET` deploy variable; existing boxes need no env change.
