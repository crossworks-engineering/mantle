# 0.83.0 — 2026-06-28

**Live streaming on by default.** Watching the assistant think and type as it
works is now the default experience, not an opt-in.

## Highlights

- **Live turn streaming enabled by default.** Agent status and tokens stream to
  the web UI as they're produced. Opt out per install with
  `MANTLE_TURN_STREAMING=0`.
- **Friendlier delegation misses** (v0.82.2). When an agent delegates to a
  slug that doesn't exist, the error now suggests the closest real agent
  instead of ballooning the turn with retries.
