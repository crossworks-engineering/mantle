# 0.20.68 — June 11, 2026

## Highlights

- **Update Mantle from the interface** (Settings → **Updates**). See the running build's identity and the latest release, then one-click "Update to vX" with live progress and an automatic reload when the new version answers. A small updater sidecar does the pull-and-restart; without it the page degrades to copy-paste CLI instructions.
- **"Update available" chip in the sidebar.** When a newer release exists, a chip appears at the top of the navigation linking straight to Settings → Updates. Up-to-date installs never see it.
- **One-line installer.** A curl-able `install.sh` fetches the deploy bundle, generates all secrets into a locked-down `.env` (zero hand-editing for localhost; set a domain for auto-HTTPS), and starts the stack. Every tagged release now builds amd64+arm64 images and publishes a matching deploy bundle automatically.
