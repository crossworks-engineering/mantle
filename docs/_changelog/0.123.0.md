# 0.123.0 — 2026-07-08

**Choose exactly what syncs from SharePoint and OneDrive.** Each enabled drive
on the Microsoft settings page now has a "Choose content" picker: browse the
drive's folders, tick the folders (everything inside them) or individual files
you want in your brain, and only those sync. Nothing ticked keeps today's
behaviour — the whole drive syncs. Changing the selection takes effect on the
next sync pass, including removing files that no longer fall inside it.

**Clearer drive discovery.** The drives list now explains what it shows: your
OneDrive plus the document libraries of SharePoint sites the connected account
*follows*. If a site is missing, Follow it in SharePoint and hit Refresh
drives — having access alone isn't enough.
