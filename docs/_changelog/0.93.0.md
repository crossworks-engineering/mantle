# 0.93.0 — 2026-06-29

**The whole brain, over MCP.** This tag carries the v0.92.0 sweep too: a
desktop Claude (or any MCP client) can now read *and write* nearly everything,
and even build mini-apps.

## Highlights

- **MCP write surface** (v0.92.0). Tables (write + query), Pages (write +
  block edits), Contacts, Workers (summarize / vision / image generation), and
  Export are all exposed as MCP tools.
- **Mini-app builder over MCP.** Create, source-edit, build, and publish `/apps`
  mini-apps from an MCP client — plus a whole-app import endpoint and an
  `apps:push` CLI for local development, and an app-authoring guide.
