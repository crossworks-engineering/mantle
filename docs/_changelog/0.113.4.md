# 0.113.4 — 2026-07-04

**The cursor shows the moment an H1 is inserted.** v0.113.3 restored the
caret's colour in gradient headings, but a just-inserted empty H1 still
collapsed to a zero-width box (`width: fit-content` with no text), so the
caret had nowhere to paint until the first letter arrived — the "typing
works but where am I?" confusion from the slash menu. The heading now keeps
a one-character minimum width, and the gradient uses a real text colour
with only the glyph fill transparent, so the caret is visible immediately.
