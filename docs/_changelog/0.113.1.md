# 0.113.1 — 2026-07-03

**Centered page title, easier to read.** The floating title in the middle of
the header now uses the app font (Inter) — smaller and bold instead of the
big script wordmark face — so longer titles fit without truncating and the
top bar reads cleaner next to a custom site name. The Bukhari script face is
now reserved for the wordmark alone.
