# 0.90.0 — 2026-06-28

**The assistant is everywhere, and agents can build their own tools.** This tag
gathers the v0.85.0–v0.89.0 run: the global assistant overlay, recipe tools,
and a full set of note/page conversion tools.

## Highlights

- **Global assistant overlay** (v0.87.0). The assistant is a content-area
  overlay on every screen — bubble or ⌘I — with a persistent thought trail.
  The dedicated `/assistant` page is retired.
- **Marker pick-mode** (v0.88.x). Point the assistant at specific rows on
  screen: pick mode highlights markable items and turns clicks into context
  chips.
- **Recipe tools** (v0.88.0/1). Agents compose existing tools into reusable
  recipes at runtime — inside a safe envelope (no shell, no privilege
  escalation), with an explicit build → approve → re-ask handoff.
- **Content conversions.** Promote a note to a page (`page_from_note`), compile
  several notes into one page (`page_from_notes`), copy a page back into an
  editable note (`note_from_page`), plus `note_from_file` and
  `page_from_journal` — round-tripping powered by a new page-doc → markdown
  serializer.
