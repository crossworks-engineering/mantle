# 0.45.0 — 2026-06-24

The foundation for pulling Microsoft 365 into the brain: delegated per-user
OAuth for Microsoft Graph.

## Highlights

- **Microsoft Graph M0 — delegated OAuth core.** The hard, net-new part landed:
  delegated per-user OAuth2 with token refresh (Mantle had no OAuth before).
  `@mantle/microsoft` holds config, an Auth-Code + PKCE flow, a sealed
  token-store with single-flight refresh (`FOR UPDATE` row lock), and a
  throttling-aware Graph client. M1–M3 (drives / mail / calendar) build on this.

## Notes

- `ms_accounts` schema + **migration 0100** (sealed tokens, AAD = row id, like
  `api_keys`). `/api/microsoft/oauth/{start,callback}` routes +
  `/settings/microsoft` UI + nav.
- **Migration 0100 is not yet applied** (`DATABASE_URL` is prod — needs a
  `pg_dump` backup + go-ahead). Needs `MS_CLIENT_ID` / `MS_CLIENT_SECRET` +
  Azure app registration to go live; until then the UI shows "Not configured".
- See [`microsoft-graph-ingest.md`](../microsoft-graph-ingest.md) for the full
  design.
