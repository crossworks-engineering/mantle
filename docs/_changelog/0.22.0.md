# 0.22.0 — June 12, 2026

## Highlights

- **Toolsmith — an agent that builds API tools from docs.** The missing half of the API Console: point Toolsmith at a service's API reference and it runs the whole loop itself — fetches the docs, authors templated HTTP tools (`{param}` placeholders + `{{secret:…}}` vault references), tests them against the live API through the real dispatcher, bundles them into a tool group, and grants the group to an agent. One prompt → a deployed ability your chats and heartbeat routines can call.
- **Three ways to use it:**
  - The **Assist panel** in the API Console (Toolsmith button), with a per-surface agent picker.
  - **Delegation** — your main assistant can hand work to Toolsmith mid-conversation.
  - **MCP** — all 12 toolsmith tools are exposed over MCP, so Claude Code / Claude Desktop can run the same loop from your own subscription.

## Notes

- Safety stances: agents author HTTP tools only (shell tools stay human-authored), tool tests refuse non-HTTP targets, and key lookups return masked references only — plaintext secrets never reach the model.
- Toolsmith ships in the system manifest; existing installs can seed it with `pnpm -C apps/web seed:toolsmith`.
- See the [API Console & Toolsmith guide](/docs/guide) for a full walkthrough.
