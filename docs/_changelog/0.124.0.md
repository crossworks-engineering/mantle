give team members the latest platform improvements in one sentence each —
SharePoint connection, the OneDrive content picker, in-place briefing reading,
checked tool use, self-healing page edits, and more — each with its own icon
and a polished staggered entrance. The strip is curated alongside releases, so
a member who only visits occasionally still catches up at a glance.
