# 0.55.0 — 2026-06-24

Filter the sidebar to the surface you want, and read every icon-rail label on
hover.

## Highlights

- **Quick filter box on the sidebar.** A sticky "Filter menu…" search box at the
  top of the expanded sidebar: case-insensitive substring match across all
  groups, empty groups drop out, Esc / × clears. Expanded-mode only (hidden at
  icon-rail width and in the collapsed rail, which always shows the full set).
- **Tooltip hints on the icon rail.** The collapsed rail now shows each item's
  label via a shadcn `Tooltip` on the right (replacing the native `title=`), with
  the badge count inline.

## Notes

- `SidebarNav` takes a `collapsed` prop (desktop passes `navCollapsed`; the
  mobile drawer stays expanded) to drive both behaviours.
- Pure UI. No schema migration in this release.
