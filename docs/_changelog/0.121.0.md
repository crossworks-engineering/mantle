# 0.121.0 — 2026-07-08

**Connecting a Microsoft account now lands you back where you started.** After
signing in with Microsoft to connect SharePoint/OneDrive, the browser is
returned to the Microsoft settings page as intended — previously the final
redirect could point at an unreachable address on some deployments, making a
successful connection look like a failure.

**Steadier page editing for assistants.** Batch page edits now report the ids
of every block they create, so multi-step restructures chain reliably instead
of tripping over stale references — and when a reference does go stale, the
error names all of them at once.
