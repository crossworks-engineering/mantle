# 0.110.0 — 2026-07-02

**Multiple admins, one brain.** A brain can now have several full-admin logins,
with a complete audit trail of who did what. (Shipped in the v0.111.0 image —
this version number was not tagged separately.)

## Highlights

- **Co-admin logins.** Settings → Users: create additional logins, reset their
  passwords, delete them. Every login is a full admin — there are no permission
  tiers; brain content stays keyed to the original (anchor) account.
- **Audit trail.** Logins, failed logins, logouts, password changes, user
  management, and every mutating API call are recorded with actor attribution —
  and the trail survives user deletion.
- **Guardrails.** The anchor account is undeletable, self-delete is blocked, and
  owner status is unreachable via the API. Deleting a login revokes its access
  on the next request.
