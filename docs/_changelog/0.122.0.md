# 0.122.0 — 2026-07-08

**Read briefings without leaving the hub.** Clicking a briefing card on the
Team Hub now opens the document right inside the hub — a reader view with a
"Back to the hub" bar — instead of jumping to a separate page. One tap back
and you're on the dashboard again; a button in the reader still opens the
document in its own tab when you want it, and ctrl/cmd-click keeps working
the way browsers always have.
