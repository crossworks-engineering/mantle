# 0.125.1 — 2026-07-09

**The hub-app builder's guide.** Documentation release: the Team Hub App SDK
doc is now a full builder's guide — the project structure to follow (thin
entry + a dedicated content file + one component per section), the designation
lifecycle, and three content-update patterns, including the zero-publish
"tiles from a Table" pattern: declare one builtin read tool and the hub's
what's-new boxes become table rows you edit and commit — live on the next hub
load with no publish at all.

## Also

- The general app-authoring guide (and its installable skill mirror) now
  cross-references the hub contract, so any agent building mini-apps knows how
  hub apps differ.
- A `manifest.hub` block is documented as reserved for future capability
  declarations — designation deliberately stays in the brain (pref + share),
  never in the app.
