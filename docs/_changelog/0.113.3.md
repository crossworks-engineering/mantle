# 0.113.3 — 2026-07-04

**You can see the cursor in an empty H1 again.** The Pages H1 gradient paints
the letters through the background (`background-clip: text`) with a
transparent text colour — and the blinking caret follows that colour, so a
freshly inserted, still-empty H1 looked focus-less: no cursor, though typing
worked. The caret is now pinned to the theme's primary colour, so an empty
heading shows a visible cursor immediately.
