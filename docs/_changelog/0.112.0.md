# 0.112.0 — 2026-07-03

**Release notes your brain can read.** The changelog joins the documentation
system as a built-in collection.

## Highlights

- **Changelog collection.** Release notes are browsable under **/docs** and,
  once the collection is enabled there, indexed by the brain — so "what changed
  in v0.99?" is answerable in chat. Ships disabled by default, like the other
  built-in collections.
- Hidden-folder semantics are preserved: `_`-prefixed folders stay out of other
  collections; the changelog is indexed only through its own collection root.
