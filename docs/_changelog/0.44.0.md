# 0.44.0 — 2026-06-24

A running app now links the host's stylesheet instead of inlining it, and
restyles live when you switch theme — no reload.

## Highlights

- **CSS via `<link>`, not inline.** The app `srcdoc` no longer inlines the
  host's whole compiled stylesheet (~386 KB) into every app; it `<link>`s the
  same already-loaded, browser-cached stylesheet files and inlines only the
  small dynamic `<style>` tags. Weather `srcdoc`: ~410 KB → 25 KB. Falls back to
  a full inline capture when no `<link>` stylesheets exist (some dev setups).
- **Live theme.** A `MutationObserver` mirrors the host `<html>` class +
  `data-color-theme` into the iframe over a new `theme` bridge message, so a
  dark/light or colour-theme switch restyles a *running* app with no reload
  (every theme's tokens are already in the linked sheet — only the selectors
  change). Verified live.

## Notes

- `style-src` is scoped to `<origin>/_next/` (an opaque origin can't match
  `'self'`).
- Completes the `lib/theme-sharing` plan begun in [0.43.0](./0.43.0.md).
- No schema migration in this release.
