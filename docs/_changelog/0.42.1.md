# 0.42.1 — 2026-06-24

One shared chat bubble, so a specialist's reply reads the same in every Assist
panel.

## Highlights

- **Universal chat bubble styling across specialist panels.** The four
  in-surface Assist chats (Apps, Pages, Tables, Dev-tools) each hand-rolled
  their own message styling — Appsmith replies at 12px, the rest at 14px, three
  different bubble looks. A new shared `components/chat-bubble.tsx` gives a
  role-based bubble (primary for the user, a bordered `bg-background` surface for
  the agent, mirroring the main `/assistant` dock) plus `CHAT_TEXT_CLASS` — a
  single 15px body size for every chat.

## Notes

- All four panels were routed through it; the per-panel bubble markup is gone.
- Pages now also tracks the picked agent name so its assistant bubbles + working
  indicator label match the others.
- Pure styling — no behaviour change. No schema migration in this release.
