# 0.111.0 — 2026-07-03

**A calmer first screen, and frontend-only development.** The right-hand
Activity column now starts hidden, and the web UI can be developed against a
deployed brain with no local stack. This is also the first deployable image
carrying v0.110.0's multi-admin logins.

## Highlights

- **Activity hidden by default.** The live Activity column starts collapsed to
  an icon rail on first visit; expand it with ⌘J or the rail toggle and your
  choice sticks.
- **Frontend-only dev: `pnpm dev:fe`.** Run just the web app against a deployed
  Mantle's API — no local Docker or Postgres. A box opts in by allowlisting the
  dev origin via `MANTLE_API_CORS_ORIGINS` (now plumbed through compose). The
  detached path was runtime-verified end to end, fixing three latent breaks
  found along the way.
