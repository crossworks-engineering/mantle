# 0.102.1 — 2026-06-30

**More reliable releases.** A continuous-integration gate now validates every
change before it can reach a deploy — so a broken build is caught automatically
instead of surfacing on a server.

## Highlights

- **CI build-check.** Every push to a feature branch, every pull request, and
  every push to `main` now runs typecheck + the full test suite + the production
  web build on GitHub — off your machine, on the right code. The image build and
  publish stay tied to a version tag as before; this is purely a pre-merge
  correctness gate that keeps `main` deployable. No user-facing behavior changes.
